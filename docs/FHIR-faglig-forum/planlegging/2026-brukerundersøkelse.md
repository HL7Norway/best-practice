# Brukerundersøkelse FFF 2026

FHIR fagforum (FFF) is an open meeting for discussing topics related to use and implementation av HL7 FHIR in Norway. FFF is open to anyone interested.

## Innledning

Vi ønsker å undersøke tilfredshet med arrangementet, og hvordan arrangementet kan forbedres til å dekke helsesektorens behov best mulig.

## Distribusjon

Foreslår å distribuere til de som deltar på møtet, samt via info@hl7.no, zulip chat og linkedin.

## Teknisk
